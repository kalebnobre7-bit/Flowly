# 🚀 GUIA DE INSTALAÇÃO - FLOWLY

## Passo a Passo pra colocar no ar:

### 1️⃣ Criar Repositório no GitHub

1. Vai em: https://github.com/new
2. **Repository name:** `flowly`
3. **Description:** "To-Do inteligente e minimalista"
4. Deixa **Public**
5. **NÃO** marca nenhum checkbox (README, .gitignore, etc)
6. Clica em **Create repository**

### 2️⃣ Fazer Upload dos Arquivos

Você tem 2 opções:

#### OPÇÃO A - Pelo navegador (mais fácil):

1. Na página do repositório vazio, clica em **uploading an existing file**
2. Arrasta os arquivos:
   - `index.html`
   - `README.md`
   - `.gitignore`
3. Escreve no commit: "🎉 Initial commit - Flowly v1.0"
4. Clica em **Commit changes**

#### OPÇÃO B - Pelo terminal (se preferir):

```bash
cd /caminho/para/flowly
git init
git add .
git commit -m "🎉 Initial commit - Flowly v1.0"
git branch -M main
git remote add origin https://github.com/SEU_USUARIO/flowly.git
git push -u origin main
```

### 3️⃣ Ativar GitHub Pages

1. No repositório, clica em **Settings** (engrenagem no topo)
2. No menu lateral esquerdo, clica em **Pages**
3. Em **Source**, seleciona **main** branch
4. Clica em **Save**
5. Aguarda 1-2 minutos
6. A URL vai aparecer: `https://SEU_USUARIO.github.io/flowly/`

### 4️⃣ Pronto! 🎉

Agora é só:

- Acessar a URL
- Usar o Flowly
- Pedir melhorias
- Eu atualizo o código
- Você dá refresh e tá atualizado!
